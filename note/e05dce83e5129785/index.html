# 简介

TypeScript是JavasSript的超集

# TypeScript的特点

TypeScript有以下几个特点

  • TypeScript在编译时会对代码进行严格的静态类型检查，将问题在编码阶段体现出来
  • TypeScript语法遵循ES规范，更新速度快，不断支持罪行的ECMAScript新特性
  • TypeScript支持面向对象的接口，抽象类，多特的特性
  • TypeScript可以提供更好的代码提示，代码补全，接口提示，跳转定义等

# 环境配置

# node安装

# 全局安装TS

执行如下命令安装

npm install -g typescript
1

安装完成后执行如下命令检查是否安装成功

tsc -v
// 正确显示下面的版本号即表示安装成功
Version 4.7.3
1
2
3

# 项目安装TS

可以单独的在项目中安装TypeScript，可以限定项目使用的TypeScript版本
执行如下命令初始化项目

yarn init -y
1

安装TypeScript

yarn add -D typescript
1

版本查看

yarn tsc -v
1

# 编译TS代码

使用tsc命令将ts代码编译为js代码，如果编译过程出错，控制台会打印错误
例如：编译hello.ts代码，会生成js代码

tsc hello.ts
1

每次修改代码都要重新手动执行命令编译ts文件，可以使用如下命令自动监听ts文件的修改并生产js文件

tsc hello.ts -w
1

# ts-node编译工具

ts-node 工具可以直接编译并执行ts文件，但是不会生成对应的js文件

全局安装命令

npm i -g ts-node
1

使用 终端中直接输入如下命令即可执行ts文件查看结果

ts-node hello.ts
1
