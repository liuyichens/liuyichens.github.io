# link 元素

link 元素是外部资源链接元素，规定了文档与外部资源的关系中

  • link 元素通常是在 head 元素中
  • 最常见的用于链接外部样式表以及 favicon 图标

# link 元素常见的属性

  • href 指定链接资源的 URL，URL 可以是相对的，也可以是绝对的
  • rel 指定链接类型
    • icon: 站点图标
    • stylesheet: CSS 样式
    • 更多类型见 https://developer.mozilla.org/zh-CN/docs/Web/HTML/Link_types
      • 例如： dns-prefetch 实验性 提示浏览器该资源需要在用户点击链接之前进行 DNS 查询和协议握手。
        对于网站的性能有一定的优化作
        使用网站： 京东
