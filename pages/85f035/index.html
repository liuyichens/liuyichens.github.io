# emmet 语法大幅度提高了前端开发效率的工具

例如

  • ! 和 html:5 可以快速生成完整结构的 HTML5 代码

# >子代和 +兄弟

div>ul>li

div+div>p>span+i

div+p+ul>li

# *多个和^上一级

ul>li*5

div+div>p>span^h1

div+div>p>span^^^^h1

# () 分组

div>(header>ul>li*2>a)+footer>p

# 属性(id 属性， class 属性， 普通属性) {} 内容

div#header+div#main>.container>a[href]

# $数字

ul>li.item$*5

# 隐式标签

.box+.container

ul>.item*3
